# design/

Ficheiros-fonte de design e variantes de marca **não usados pelo site**
(PDFs de logótipo, exports antigos de hero, variantes de logo).

- Nada aqui é referenciado por HTML/CSS/JS — é apenas arquivo.
- Imagens servidas pelo site vivem em `img/` (globais) e `card/` (cartões NFC usados na landing).
- Antes de apagar ou mover algo daqui para `img/`, confirmar com
  `grep -r "<nome-do-ficheiro>"` que continua sem referências.
